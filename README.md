# Business Context

The credit operation required manual generation of financial and operational reports, consuming several hours daily and increasing operational risk.

Main problems identified:

- Manual report generation
- High operational effort
- Lack of standardized dashboards
- Delayed KPI updates
- No automated export process

---

# Solution Developed

An end-to-end automation architecture was created using Excel VBA.

The solution performs:

- Automatic KPI calculations
- Credit portfolio analysis
- Regional performance monitoring
- Dashboard auto-refresh
- PDF export automation
- CSV export automation
- Automation logging
- Version control
